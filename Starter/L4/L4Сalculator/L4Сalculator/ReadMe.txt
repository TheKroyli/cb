+ Используя Visual Studio, создал проект по шаблону Console Application, называем его "L4Calculator".
   + Создаем две переменние op1  и op2 (произвольные числа), для приема ввода от пользователя используем строковою переменную sigh.
   + Для обработки вычесления использую перемунную decision.
   + Для выбора операции исполузуется переключатель switch c 4 кейсами.
   + Для запрета деления на 0, использовал операторов ветвления if-else.
