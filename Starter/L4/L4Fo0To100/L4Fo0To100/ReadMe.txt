+ Используя Visual Studio, создал проект по шаблону Console Application.
   + Создаем переменную number1 типа string, и с помощь операции Convert переводим ее в int (под названием Number).
   + Для организации выбора использовал if-else.
   + Программа анализирует ввод с клавиатури и выдает сообщение в каком числовом промежутке было число (от 0 до 14, от 15 до 35, и тд.).
   + Если число ввода больше 101, то получим следующее сообщение о том что число не входит не в какой числовой промежуток.
