+ Используя Visual Studio, создайте проект по шаблону Console Application, называем его "CalculatorApp".
   + Создаем две переменние operand1  и operand2 (произвольные числа), для приема ввода от пользователя используем строковою переменную sigh.
   + Так же string operand1 и string operand2 конверкируем в op1 и op2 (делаем из string -> int).
   + Для обработки вычесления использую перемунную decision.
   + Для выбора операции исполузуется переключатель switch c 4 кейсами.
