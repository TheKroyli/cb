+ Используя Visual Studio, создал проект по шаблону ConsoleApplication. 
   + В программе присутствует 2 варианта решения.
    + Метод №1
	   + Мы знаем что число нечетно, когда его младший бит равен 1, и четно, когда младший бит равен 0.
	   + Приравняем наше число при помощи побитового И к 1 (…000001), если выражения result == 0 то число четное, если result = 1 - то нечетное.

	+ Метод №2
	   + Выполняем операцию остатка от деления на 2, и если результат равен 0, то выведем на екран 'Число четное'.
	     Если результам не 0 - 'Число нечетное'.
